Visual Impairments

According to the World Health Organisation (WHO), there are around 2.2 billion people globally who have a near or distance vision impairment. They estimate that everyone, if they live long enough, will develop at least one eye condition in their lifetime. The risk of developing a visual impairment increases depending on your age, whether you have genetics that can cause a visual impairment, your ethnic background, your lifestyle and the factors in your environment.

This is a large amount of people who can't potentially access or understand websites with poor accessiblilty coding.

Web Content Accessibility Guidelines

Web Content Accessibility Guidelines (WCAG) is a set of guidelines proposed by the Web Accessibility Initiative (WAI) which is an intitiative made by the World Wide Web Consortium (W3C) to develop and improve accessiblilty online for people with all types of impairments. WCAG is a set of standards and guidelines for accessiblilty for web content for people with disabilities and who need to use assistive technologies, such as screen readers, to make web content more accessible to people.

WCAG is split up into four categories that majorly affect how accessible websites are:

  • Percievable: Users must be able to percieve the web content using one or more of their senses.
  • Operable: Users must be able to control interactable web elements in some way.
  • Understandable: Users must be able to understand the content.
  • Robust: The content must be developed using adopted web standards that will work on different browsers now and in the future.

By following WCAG, we can effectively improve our website to become more accessible along with making content more friendly to assistive technologies, like screen readers.

Making websites more accessible

We can make more accessible by making content more friendly to screen readers and other assistive technologies and using HyperText Markup Language (HTML) and Cascading Style Sheets (CSS) in a way to improve accessibility, and creating websites with accessibility in mind. We can also use WCAG as a set of standards and as a guide to further improve accessibililty on websites.
